## Project Overview

This project develops a predictive machine learning model to estimate household income across 
32,526 US neighborhoods based on geographic and demographic characteristics. Using advanced 
ensemble methods and comprehensive feature engineering, we achieve an 88% accuracy rate 
(R² = 0.88) in predicting mean household income. The model leverages state-level geographic 
data, population density metrics, and income distribution patterns to provide reliable income 
predictions at the census tract level. This tool enables real estate investors, urban planners, 
policymakers, and businesses to make data-driven decisions regarding market expansion, community 
development, and economic analysis.
